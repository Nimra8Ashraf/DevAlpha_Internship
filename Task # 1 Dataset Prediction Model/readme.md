# 📈 Stock Price Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting future stock closing prices using Machine Learning regression techniques. Historical stock market data is analyzed, preprocessed, and used to train a predictive model capable of estimating future stock prices.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

---

## 🎯 Objective

The objective of this project is to build a regression model that predicts stock closing prices based on historical market data.

---

## 📂 Dataset

The dataset contains historical stock market information, including:

- Date
- Open Price
- High Price
- Low Price
- Close Price (Target)
- Adjusted Close
- Trading Volume

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yFinance (for downloading stock data)

---

## 📊 Project Workflow

1. Import required libraries
2. Load historical stock data
3. Perform Exploratory Data Analysis (EDA)
4. Handle missing values
5. Select relevant features
6. Split dataset into training and testing sets
7. Train the regression model
8. Predict stock prices
9. Evaluate model performance
10. Visualize actual vs predicted prices
