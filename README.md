# ЁЯУИ Task 1: Stock Price Prediction (Dataset Prediction Model)

## Objective

Develop a regression model to predict future stock closing prices using historical stock market data.

## Project Description

This project uses historical stock market data to train a Machine Learning regression model capable of predicting stock closing prices. The workflow includes data preprocessing, visualization, feature selection, model training, and evaluation using regression metrics.

## Dataset Features

- Date
- Open Price
- High Price
- Low Price
- Close Price (Target)
- Adjusted Close
- Volume

## Workflow

- Import required libraries
- Load stock market dataset
- Perform Exploratory Data Analysis (EDA)
- Handle missing values
- Select relevant features
- Split the dataset into training and testing sets
- Train the regression model
- Predict stock prices
- Evaluate model performance
- Visualize actual vs predicted prices

## Machine Learning Algorithm

- Linear Regression

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R┬▓ Score

# тЭдя╕П Task 2: Heart Disease Prediction (Classification Project)

## Objective

Build a Machine Learning classification model to predict whether a patient is likely to have heart disease based on medical attributes.

## Project Description

This project analyzes patient health records and develops a classification model for heart disease prediction. Multiple classification algorithms are trained and compared to identify the best-performing model.

## Dataset Features

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease)

## Workflow

- Import required libraries
- Load the dataset
- Perform Exploratory Data Analysis (EDA)
- Handle missing values
- Encode categorical variables
- Scale numerical features
- Split the dataset into training and testing sets
- Train multiple classification models
- Predict patient outcomes
- Evaluate and compare model performance

## Machine Learning Algorithms

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Evaluation Metrics

- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix
